Job Tracker API Documentation

Welcome to the Job Tracker API

The Job Tracker API provides access to job listings scraped from CareerBeacon, specifically for New Brunswick, Canada. Through our API endpoints, users can retrieve up-to-date job postings, including details such as job titles, company names, locations, descriptions, and application links. This API is designed for developers, job seekers, and businesses who want to programmatically access job data without manually browsing job boards.

Below, you'll find a list of available endpoints, including details on how to authenticate and retrieve job listings.

Endpoints

  • GET /api/jobs/ - Retrieve all jobs.
  • GET /api/new_jobs/ - Retrieve all new jobs (posted in the last week).
  • GET /api/<city>/ - Retrieve jobs in a specific city.
  • GET /api/new/<city>/ - Retrieve new jobs in a specific city.
  • GET /api/title/<jobType>/ - Retrieve jobs by job type.
  • GET /api/title/<jobType>/<city>/ - Retrieve jobs by job type in a specific city.
  • GET /api/title/new/<jobType>/<city>/ - Retrieve new jobs by job type in a city.

Models

Job Model

Field Type Description
title CharField (max_length=200) The job title.
link URLField (max_length=300) URL to the job posting.
company CharField (max_length=200) The company offering the job.
location CharField (max_length=100) Location of the job.
date_posted DateField (nullable) The date the job was posted.
job_details TextField (max_length=500) Brief job details.
description TextField (max_length=1000) Full job description.
created_date DateTimeField (auto_now_add=True) The date the job was added to the database.

Authentication: The API requires authentication (Basic Auth). Account creation and profile set up will need to be done to access job alerts and view the API data
